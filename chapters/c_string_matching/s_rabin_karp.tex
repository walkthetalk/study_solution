 Alice 和 Bob 都希望直到他們的文件是否一樣。
爲了避免傳送整個文件 A 或 B，
他們運用下列快速的概率檢查方法。

並從 \m{\{0,1,\ldots,q-1\}} 中隨機選取一個整數 \m{x}。

\startformula
A(x)=\left(\sum_{i=0}^{n-1} a_i x^i \right)\mod q
\stopformula

則 \m{A(x)=B(x)} 的概率至多爲 \m{1/1000}；
如果兩個文件相同，則 \m{A(x)} 的值必定等於 \m{B(x)} 的值。
