 \m{c} 為壹個 \m{n} 維向量。
那麽 Farkas 引理說明正好有壹個系統
\startformula\startmathalignment
\NC Ax \NC \le 0 \NR
\NC c^T x \NC > 0 \NR
\stopmathalignment\stopformula
以及
\startformula\startmathalignment
\NC A^T y \NC = c \NR
\NC     y \NC \le 0 \NR
\stopmathalignment\stopformula
是可解的，其中 \m{x} 是壹個 \m{n} 維向量，
 \m{y} 是壹個 \m{m} 維向量。
證明 Farkas 引理。
